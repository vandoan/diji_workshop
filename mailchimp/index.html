Mailchimp away

Mailchimp is no joke. Building a responsive Mailchimp template can drive a person to insanity. Here are some issues that you don't realize before starting.

- The type of coding is like early 90's: tables, rows, columns.
- Try making that responsive.
- Test on so many email clients: Apple Mail, Apple Mail mobile, Yahoo, Gmail, Gmail app, MS Outlook (notorious), MS Outlook web account.
- To send tests templates, you have to go through Mailchimp.com.
- Automating it through a backend server is more intense.
- Careful, or you can send a test email to all your 40k or how many ever on that list.
- Mailchimp code compilation can be quirky as hetch.
- And the list goes on.
